# AMSelect — LPBF Machine & Powder Advisor

A single-file web app for selecting laser powder bed fusion (LPBF) printers and metal powders.

- **46 powders** in 9 material categories with **529 supplier listings** (verified July 2026)
- Guided machine selector, powder library with per-supplier prices, US supplier search
- Company directory (58 companies), RFQ request builder with custom-powder support, supplier feedback portal

Open `index.html` in any browser — no build step, no dependencies.

Prices and specifications were collected from supplier websites in July 2026 and change with commodity markets. Always confirm with the supplier before ordering.
